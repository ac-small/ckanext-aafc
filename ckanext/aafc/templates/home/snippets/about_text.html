The AAFC Data Catalogue is a metadata management system which supports Open Government by providing AAFC with the ability to inventory, register and publish the metadata about AAFC’s current data and information holdings.

Who can use it?

The AAFC Data Catalogue is an environment for everyone who wants to use, publish, share and open data, or those who simply want to know more about using data.

Sharing Data Internally and Open Government

The data sharing guide provides an overview for data stewards and the community of the key principles of data sharing within AAFC and externally to the Open Government Portal.

A critical support for government data sharing is public trust in how government uses and shares the data we collect. This data sharing guide outlines:

  • data sharing principles
  • data sharing checklist
  • making data safe for sharing
  • making data available for sharing
  • designing and managing data for sharing
  • data sharing agreements and licenses
  • ready to share data with AAFC and/or Open Government
  • requesting access to data
  • responding to data access requests

Get involved

The AAFC Data Catalogue is a living program and it needs you to continuously improve and to become the environment that AAFC and its collaborators need it to be.

You can get involved by:

  • providing feedback on any part of the AAFC Data Catalogue content and environment
  • telling us about functionality you’d like to see
  • suggesting guidance you’d like to see and help us write it
  • sharing resources you’ve developed that we can incorporate
  • contributing data to the AAFC Data Catalogue
  • suggesting your work on how you’re using data as a case study
  • collaborating with other data Jedi’s and Yoda’s in the data community
